# Physik-Web-Apps (Alpha)

Willkommen im Repository für interaktive Physik-Web-Apps. Diese Tools sind speziell für den Einsatz im Physikunterricht konzipiert und befinden sich aktuell in der **Testphase (Alpha-Version)**.

> [!IMPORTANT]
> **Status:** Alpha-Version. Die Apps sind funktional, können aber noch Fehler enthalten oder in der Darstellung variieren. Vor dem produktiven Einsatz im Unterricht wird eine Begutachtung und Testung dringend empfohlen.

## 🚀 Live-Ansicht
Die Apps können direkt über den Browser aufgerufen werden:
👉 **[https://mmiedl811.github.io/Physik-Web-Apps/](https://mmiedl811.github.io/Physik-Web-Apps/)**

## 📱 Zielplattform
Die Anwendungen sind primär für die Nutzung auf dem **iPad** optimiert (Touch-Steuerung, Viewport-Anpassung), um einen reibungslosen Workflow im digitalen Unterricht zu gewährleisten.

## 🛠 Technik & Konzept
Die Apps basieren auf nativem Web-Stack (HTML5, CSS3, JavaScript). 
- **Didaktische Reduktion:** Fokus auf physikalische Kernkonzepte.
- **Interaktivität:** Parameteränderungen in Echtzeit.
- **No-Install:** Sofort einsatzbereit ohne Registrierung oder App-Store.
